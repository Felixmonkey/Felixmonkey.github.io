关机

shutdown 参数 时间
option:
　　-c: 取消前一个关机命令
　　-h: 关机
　　-r: 重启(reboot)

系统默认运行级别

cat /etc/inittab: 修改系统默认运行级别(id:3:initdefault:)
runlevel: 查看系统运行级别
logout: 退出登录命令（推荐这习惯）

系统设备挂载

mount: 查询系统中已经挂载的设备
mount -a：依据配置文件/etc/fstab的内容，自动挂载
挂载光盘:
　　mkdir /mnt/cdrom/: 建立挂载点(mnt目录可换成media 、misc)
　　mount /dev/cdrom /mnt/cdrom/: 挂载光盘
　　= mount -t iso9660 /dev/sr0 /dev/cdrom/
umount 设备文件名或挂载点：卸载(注意当前路径不能在cdrom目录中)
umount /mnt/cdrom
fdisk -l: 查看U盘/硬盘设备文件名

查看登录用户信息

w、who：查看登录用户信息
last: 查询当前登录和过去登录的用户信息(默认读取/var/log/wtmp文件数据)
lsatlog: 查询所有用户登录时间

输出命令echo

echo: 输出命令
echo -e: 可识别控制字符
echo -e \: 按照16进制同样可以输出
　　\e[1;31m : 开启颜色显示（31m为红色）
　　\e[0m: 关闭颜色

脚本执行

#在linux中表示注释（但是在脚本中#!/bin/bash必须写入）
脚本执行(两种方法):
　　(1). chmod 755 脚本文件 然后执行文件
　　(2). bash 脚本文件（可以不赋权限）
