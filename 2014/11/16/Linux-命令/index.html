Linux 命令

Contents
  1. 1. 文件处理命令
  2. 2. 压缩命令

文件处理命令

ls -a: 显示文件包括隐藏文件
ls -l: 显示文件详细信息 == ll
ls -d: 显示目录
mkdir -p xxx/xxx: 创建多层目录(没有参数p，则只能创建一个)
touch: 创建文件
cat: 查看文件内容
cd: change directory
cd .: 当前目录
cd ..: 上一级目录
cd -: 上一次目录
ctrl l: 清屏 == clear
pwd: print working directory
rm -rf 文件或目录: 删除文件或目录(注意也可删除/)
mv: 文件改名/移动(同一目录下/非同一目录下)

压缩命令

zip: 压缩文件名 源文件 #压缩文件
zip -r: 压缩文件名 源目录 #压缩目录
unzip 文件或文件夹: 解压缩.zip文件
gzip 源文件:压缩为.gz格式的压缩文件，源文件会消失
gzip -c 源文件 > 压缩文件: 则会保留源文件
gzip -r 目录: 压缩目录下的所有子文件，但不能压缩目录
gzip -d 压缩文件: 解压缩
gunzip: 解压缩
bzip2 源文件:压缩为.bz2格式的压缩文件，源文件会消失 #bzip2不能压缩目录
bzip2 -k 源文件: 保留源文件
bzip2 -d 压缩文件: 解压缩，-k保留压缩文件
bunzip2 压缩文件: 解压缩，-k保留压缩文件

tar -cvf 打包文件名 源文件
‘option’：
-c: 打包
-v: 显示过程
-f: 指定打包后的文件名

tar -xvf 打包文件名
‘option’：
-x: 解打包
其实.tar.gz格式是先打包为.tar格式，再压缩为.gz格式

tar -zcvf 压缩包名.tar.gz 源文件
‘option’：
-z: 压缩为.tar.gz格式

tar -zxvf 压缩包名.tar.gz
‘option’：
-x: 解压缩为.tar.gz格式

tar -jcvf 压缩包名.tar.bz2源文件
‘option’：
-z: 压缩为.tar.bz2格式

tar -jxvf 压缩包名.tar.bz2源文件
‘option’：
-x: 解缩为.tar.bz2格式
