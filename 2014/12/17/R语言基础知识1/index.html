对象类型

R中有五类最基本的对象类型：
character :字符型
numeric :(real numbers) 数值型：实数或小数
integer :整型
complex :复数型
logical :(True/False) 逻辑型

对象Vector

一个vector中只能包含同类型的对象(除了list，它可以包含各种类型的对象)
vector（）函数有两个basic arguments，第一个是vector中的对象类型，第二个是vector的长度。
numbers在R中被认为是numeric类型，若明确想要integer类型，则在numbers后加上L.

Matrices

Matrices是一种具有维度属性的vector,维度为integer型，大小为2，分别为矩阵的行数和列数

Factors

因子记录分类属性的数据，使用有标签的factor要优于整型向量，更有描述性

data frame

数据框储存表格数据，它可以将几个不同类型但长度相同的向量用data.frame（）函数合并到一个数据框，它的模样就像二维数组。但要注意：合并的几个向量长度必需一致。每一列数据类型可以不同，可以通过data.matrix（）将数据框转换为矩阵。

Names

特殊值数据

为确保所有数据都能被正确识别、计算或统计等，R定义了一些特殊值数据：
NULL：空数据
NA：表示无数据
NaN：表示非数字,代表未定义的值，即not a number
一个NaN的值也是NA，但是反过来不成立
inf：数字除以0得到的值,无穷大
判断数据是否为这些特殊数据
is.null(x) is.na(x) is.nan(x) is.infinite(x)

获取数据类型信息的一些有用函数

getClass( )函数我们前面已经见过了，它的参数是表示类的字符串。
class( )可获取一个数据对象所属的类，它的参数是对象名称。
str( )可获取数据对象的结构组成，这很有用。
mode( )和storage.mode( )可获取对象的存储模式。
typeof( )获取数据的类型或存储模式。
